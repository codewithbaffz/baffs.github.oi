# ProductivityApp (fresh scaffold)

This workspace contains a fresh scaffold of the original `schedulfy` app with the same high-level architecture: a `schedulfy` package using Vite + React + Tailwind, plus organized `src/` with `components/`, `pages/`, `api/` and `lib/`.

To run:

```bash
cd schedulfy
npm install
npm run dev
```

I'll continue porting more modules and tests on request.
